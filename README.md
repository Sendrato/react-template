# Sendrato React template

It is a template for the fast development of the Front end.

## Installation

```bash
yarn install
```

## Development

```bash
yarn app:start:dev
```

## Build

```bash
yarn app:build
```

## Lint

```bash
yarn lint
```

## Storybook

Move into the app directory

``bash
cd packages/app
```

And run storybook local

```bash
yarn storybook
```

Open the storybook locally at http://localhost:6006
