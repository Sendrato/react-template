# Sendrato Cashless Frontend

## Development

Install project dependencies with Yarn: `$ yarn`

### Run app

```sh
$ yarn app:start:dev
```

### Linting

Project uses ESLint, stylelint and commitlint:

```sh
$ yarn lint
```

Format code with Prettier:

```sh
$ yarn prettier
```

### Docker

Create and run a Docker image

```sh
$ yarn docker
```
